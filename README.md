# Metis-Elec

Repositories pour de le code de l'équipe électrique les trois équipes sont : 

- Acquisiton 
    - Capter les signaux EMG du patient pour le transmettre au Jetson

- Moor_Control
    - Contrôle de la position de tous les moteurs pour ateindre la position désiré

- BLDC_Driver
    - Contrôle individuelle de chaque moteur BLDC pour qu'il soit à la position recquise.
